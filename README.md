# Involved Sectors
Technologies:
* Java 17
* Spring Boot
* Hibernate
* H2 Database
* JUnit + Mockito
* Python
* Bootstrap
* JavaScript

## How to run?
```sh
docker compose up -d
```
Python script in `Dockerfile` will create `data.sql` for Gradle build and run application on port 8080.

## Ports
8080 for Back-End, Front-End stored as a static content
